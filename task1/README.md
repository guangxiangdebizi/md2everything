# 密码学算法实验系统

## 项目简介

本项目实现了三种经典密码学算法的加密解密和哈希计算功能：
- **DES (Data Encryption Standard)** - 对称加密算法
- **RSA (Rivest-Shamir-Adleman)** - 非对称加密算法  
- **SHA-1 (Secure Hash Algorithm 1)** - 哈希算法

## 功能特点

✅ **图形化界面** - 基于Tkinter的友好操作界面  
✅ **DES加密解密** - 支持自定义密钥的DES加解密  
✅ **RSA加密解密** - 支持多种密钥长度(1024/2048/3072/4096位)  
✅ **SHA-1哈希** - 计算消息摘要值并验证一致性  
✅ **实时验证** - 所有操作都有即时反馈和错误提示

## 使用说明

### DES加密解密

1. 在"DES加密解密"选项卡中输入8字节密钥
2. 输入要加密的明文
3. 点击"加密"按钮生成密文
4. 点击"解密"按钮还原明文

### RSA加密解密

1. 在"RSA加密解密"选项卡中选择密钥长度
2. 点击"生成密钥对"按钮
3. 输入要加密的明文（注意长度限制）
4. 点击"公钥加密"进行加密
5. 点击"私钥解密"进行解密

### SHA-1哈希计算

1. 在"SHA-1哈希"选项卡中输入消息
2. 点击"计算哈希值"按钮
3. 查看160位的哈希值结果

## 算法说明

### DES算法
- **类型**: 对称加密算法
- **密钥长度**: 56位有效位 (64位含校验位)
- **分组长度**: 64位
- **加密模式**: ECB模式
- **填充方式**: PKCS7填充

### RSA算法
- **类型**: 非对称加密算法
- **密钥长度**: 1024/2048/3072/4096位可选
- **加密方式**: PKCS#1 OAEP
- **应用场景**: 小数据量加密、数字签名

### SHA-1算法
- **类型**: 哈希算法
- **输出长度**: 160位 (40个十六进制字符)
- **特点**: 单向、抗碰撞、雪崩效应
- **应用场景**: 数据完整性校验、数字签名

## 注意事项

⚠️ **RSA加密长度限制**: RSA加密有明文长度限制，对于2048位密钥，最大明文长度约为214字节

⚠️ **SHA-1安全性**: SHA-1已被证明存在碰撞漏洞，实际应用建议使用SHA-256或SHA-3

⚠️ **DES安全性**: DES密钥长度较短，现代应用建议使用AES
